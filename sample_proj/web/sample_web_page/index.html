AdBrix Remaster(v1.0.2)

AdBrix Remaster Hybrid Sample

애드브릭스 리마스터를 하이브리드 앱에 연동하는 기본 샘플입니다.

주요 API

1.구매

앱 내에서 구매가 발생했을 때에 purchase api를 이용하여 구매가 발생한 상품의 상세 정보가 포함된 이벤트를 전달합니다.

2.상품상세 보기

고객이 특정 상품의 상세 정보가 있는 상세 페이지로 진입했을 때 productView api를 이용하여 상품의 상세 정보가 포함된 이벤트를 전달합니다.

3.장바구니 담기

고객이 특정 상품을 장바구니에 담았을 때에 addToCart api를 이용하여 장바구니에 담긴 상품 상세 정보가 포함된 이벤트를 전달합니다.

4.상품리스트 보기

고객이 상품 리스트를 보았을 때에 listView api를 이용하여 노출된 상품 리스트 정보가 포함된 이벤트를 전달합니다.

5.관심상품 추가

고객이 특정 상품을 관심상품으로 추가했을 때에 addToWishList api를 이용하여 관심상품으로 추가된 상품의 정보가 포함된 이벤트를 전달합니다.

6. 공유하기

고객이 특정 상품을 공유했을 때에 share api를 이용하여 공유된 상품의 정보가 포함된 이벤트를 전달합니다.

AdBrix Remaster

구매하기

고객이 상품을 구매했을 때, 구매한 상품의 상세 정보를 포함하여 구매 이벤트를 전달합니다.

purchase api 를 이용하여야 하며 주문번호, 상품아이디, 상품명, 상품단가, 구매수량, 상품 카테고리 정보를 필수로 전달하여야 합니다.

본 샘플에서는 adbrix://purchase? 와 같은 JavaScript Interface를 구성하여 데이터를 전달합니다.

아래 데이터 중 * 표시가 된 것은 필수 전달 데이터 항목입니다.
